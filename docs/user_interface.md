## **1. 设计总则**

*   **核心目标：** 围绕“评测-学习-再评测”核心学习闭环，打造极简、高效、沉浸式用户体验。
*   **设计原则：**
    *   **极简高效：** 界面元素精简，核心操作一步直达，避免信息过载。
    *   **流程驱动：** “今日学习”Tab是绝对核心，界面设计严格遵循三阶段流程，引导用户自然前进。
    *   **状态明确：** 通过视觉状态清晰地展示每个阶段的“待开始”、“进行中”和“已完成”，并用锁图标🔒强制顺序依赖。
    *   **数据透明：** 关键学习成果（如新学/复习词数、等级提升）在完成时清晰反馈。

---

## **2. 核心流程：注册与登录**

**目标：** 让新用户在30秒内完成所有必要配置，直接进入核心学习流程。

### **2.1 流程概览**
采用三步引导式设计，分步收集用户信息，降低认知负担。

### **2.2 详细页面**

*   **第一步：手机号验证**
    *   **标题：** 欢迎！开启你的高效背词之旅
    *   **输入框 1：** `手机号` (占位符：请输入11位手机号)
    *   **按钮 1：** `获取验证码` (位于输入框右侧)
    *   **输入框 2：** `短信验证码` (占位符：请输入6位验证码)
    *   **主按钮：** `下一步` (手机号和验证码验证通过后激活)

*   **第二步：基础档案设置**
    *   **标题：** 完善你的学习档案
    *   **进度条：** `步骤 2/3`
    *   **输入框：** `昵称 (选填)` (占位符：输入昵称，或留空由系统生成)
    *   **选择器：** `我是` → 选项：`学生` (默认) / `家长`
    *   **选择器：** `我的英语水平` → 选项：`零基础` / `小学` (默认) / `初中` / `高中`
        *   *（若选“小学”，动态弹出子选择器：一年级 至 六年级）*

*   **第三步：学习偏好配置**
    *   **标题：** 定制你的学习计划
    *   **进度条：** `步骤 3/3`
    *   **卡片组 (单选)：**
        *   `轻松模式`：10-15分钟 | 5词+1组+1句
        *   `正常模式`：15-20分钟 | 7词+1组+1句 **(默认选中)**
        *   `努力模式`：20-25分钟 | 10词+2组+1-2句
    *   **开关：** `启用拼写测试` (默认：关闭)
    *   **选择器：** `发音偏好` → 选项：`英式发音` (默认) / `美式发音`
    *   **主按钮：** `完成，开始学习！` (点击后直接跳转至“今日学习”Tab)

## **4. 核心功能详解**

### **4.1 今日学习 (📚)**

**核心目标：** 引导用户完成“前置评测 → 学习阶段 → 当日评测”三步流程，严格保证顺序依赖。

#### **4.1.1 主页面设计**

*   **页面结构：** **三阶段流程卡片** + **顶部进度条**
*   **顶部区域：**
    *   **进度条：** “今日目标：7个单词 + 1词组 + 1句型”
    *   **模式标识：** “当前模式：正常模式”
*   **中部核心区域 (三张大卡片，纵向排列)：**

    *   **卡片 1: 前置评测**
        *   **标题：** “第一步：摸底评测”
        *   **副标题：** “测一测还记得多少？”
        *   **状态：** “待开始” / “进行中... 3/7” / “已完成 ✅”
        *   **操作按钮：** “开始评测”
        *   **交互：** 点击后，**全屏进入“前置评测”子流程**。

    *   **卡片 2: 学习阶段** (前置评测完成后自动解锁)
        *   **标题：** “第二步：针对性学习”
        *   **副标题：** “根据你的水平定制学习内容”
        *   **状态：** “等待中... (🔒)” / “进行中... X/Y” / “已完成 ✅”
        *   **操作按钮：** “开始学习” (前置评测未完成时，按钮置灰且带锁图标)
        *   **交互：** 点击后，按单词等级（L0-L4）学习。

    *   **卡片 3: 当日评测** (学习阶段完成后自动解锁)
        *   **标题：** “第三步：巩固验收”
        *   **副标题：** “检验今日学习成果！”
        *   **状态：** “等待中... (🔒)” / “进行中... X/Y” / “已完成 ✅”
        *   **操作按钮：** “开始验收” (学习阶段未完成时，按钮置灰且带锁图标)
        *   **交互：** 点击后，**全屏进入“当日评测”子流程**。

*   **流程完成反馈：**
    *   当“当日评测”完成后，**自动弹出半屏总结卡片**。
    *   **内容：**
        *   `新学单词：3个`
        *   `复习单词：4个`
        *   `等级提升：2个`
        *   `整体正确率：85%`
    *   **按钮：** “太棒了！” (关闭弹窗)

#### **4.1.2 评测子流程设计 (前置评测 & 当日评测)**

*   **通用框架 (每个单词)：**
    *   **顶部 (10%)：** `单词 3/7` + `⏸️` (暂停按钮)
    *   **中部 (80%)：** **动态渲染评测内容** (根据单词历史记录或当前等级)
    *   **底部 (10%)：** **动态操作按钮** (如“提交”、“开始录音”)

*   **前置评测题型 (根据历史记录)：**
    *   **首次评测：** 听单词 → (若错) 英译中 → (若对且启用) 拼写测试。
    *   **历史=[L3]：** 英译中测试。
    *   **历史≥3次且近2次≥L2：** 听单词 → (若错) 英译中。
    *   **历史[0]≤L1且近2次≥L2：** 中译英 → 读单词测试。
    *   **其他情况：** 英译中 → 读单词测试。

*   **当日评测题型 (根据学习后等级)：**
    *   **L0-L1：** 词义匹配 (中文6选1)。
    *   **L2：** 发音跟读评估 + 词义匹配。
    *   **L3：** 拼写测试 + 听力理解 (语境题)。
    *   **L4：** 快速反应 (1.2倍速) + 综合应用 (造句)。

*   **统一交互规则：**
    *   **即时反馈：** 操作后立即显示 `✅ 正确！` 或 `❌ 再试试`。
    *   **自动跳转：** 反馈后1.5秒，**自动进入下一个单词**。
    *   **流程结束：** 所有单词完成后，**自动返回“今日学习”主页面**。

---

### **4.2 快速复习 (⚡)**

*   **主页面：**
    *   **超大按钮：** “立即开始快速复习”
    *   **副标题：** “今日待复习：15个单词”
*   **复习子流程：**
    *   **界面：** 全屏显示一个单词 + 6个中文选项。
    *   **发音：** 自动播放一次正常语速发音。
    *   **操作：** 点击选项 → **自动进入下一题**。
    *   **计时：** 每题限时10秒。

---

### **4.3 专项训练 (🎯)**

*   **页面结构：** **功能入口网格 (2x2)**
*   **顶部标题：** “专项能力训练”
*   **功能入口：**

    | 入口名称 | 图标 | 描述 | 交互 |
    | :--- | :--- | :--- | :--- |
    | **拼写大师** | ✍️ | 听音辨词，攻克拼写难关 | 点击进入拼写训练 |
    | **听力达人** | 👂 | 磨耳朵，提升辨音能力 | 点击进入听力训练 |
    | **逻辑拓展** | 🧩 | 词义辨析、语境应用 | 点击进入逻辑训练 |
    | **发音评测** | 🎤 | AI智能纠音，说地道英语 | 点击进入发音训练 |

---

### **4.4 我的 (👤)**

*   **页面结构：** **数据看板** + **精简功能列表**
*   **顶部数据看板：**
    *   **卡片 1: 本周成就**
        *   学习天数：`7天`
        *   学习单词：`120个`
    *   **卡片 2: 能力雷达图**
        *   显示“听力、发音、拼写、词义、应用”五维能力评分。
*   **中部功能列表：**
    *   `📊 学习报告` (跳转至详情页)
    *   `⚙️ 学习设置` (弹窗：可修改模式、发音偏好、拼写开关)
    *   `📝 英语水平` (弹窗：可重新评估)
    *   `👨‍👩‍👧‍👦 家长中心` (仅家长角色显示)
*   **底部区域：**
    *   `❓ 帮助与反馈`
    *   `🚪 退出登录`

## **5. 视觉与交互规范**

*   **色彩体系：**
    *   **主色调：** 宁静蓝 (#4A90E2) - 用于主要按钮和进度条。
    *   **辅助色：** 成功绿 (#4CAF50)，错误红 (#F44336)。
    *   **等级色：** L0(红)→L1(橙)→L2(黄)→L3(浅绿)→L4(深绿) - **仅在学习/评测详情页使用**。
*   **动效规范：**
    *   **页面切换：** 平滑左右滑动。
    *   **状态变化：** 卡片解锁时轻微“弹出”效果。
    *   **成就反馈：** 完成流程时播放简洁粒子庆祝动画。
*   **字体规范：**
    *   **系统字体：** iOS (San Francisco), Android (Roboto)。
    *   **核心内容：** 单词拼写使用 **大号、加粗** 字体，确保清晰易读。
